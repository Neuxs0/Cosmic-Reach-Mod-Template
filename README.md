# Mod Template
A personalized mod template

## Dependencies:
- Cosmic Reach
- Puzzle Loader

## How to use
- Run the mod as any other Puzzle mod

## How to build
Requirements:
- JDK 24 (You can install it [here](https://www.oracle.com/java/technologies/javase/jdk24-archive-downloads.html).)
- Python 3 (You can install it [here](https://www.python.org/downloads/).)

1. Run `./gradlew build`
3. The jar will be in the newly generated `build/libs/` folder
